# ACA - Grammar-Constrained GPU Kernel Generation

Bienvenido al material de lecturas del curso **ACA** del Tecnológico de Monterrey.

## Descripción del Curso

Este curso explora la intersección entre **Inteligencia Artificial**, **Teoría de Compiladores** y **GPU Computing** para generar kernels GPU de alto rendimiento usando modelos de lenguaje con restricciones gramaticales.

## Módulos del Curso

El curso consta de 6 módulos interrelacionados, cada uno con 8-10 semanas de lecturas:

### Fundamentos Teóricos

1. **[Inteligencia Artificial](ai/index.md)** - Fundamentos de IA, Deep Learning, Transformers y LLMs
2. **[Compiladores](compilers/index.md)** - Teoría de lenguajes formales, gramáticas y parsing
3. **[Estadística](stats/index.md)** - Análisis estadístico riguroso para investigación

### Habilidades de Investigación

4. **[Research](research/index.md)** - Metodología de investigación y escritura de tesis

### Proyectos Aplicados

5. **[Project 1: LLM Code Generation](project_1/index.md)** - Generación de código con LLMs y restricciones gramaticales
6. **[Project 2: GPU Computing](project_2/index.md)** - Programación GPU con CUDA y Triton

## Estructura Pedagógica

```
Semana 1-3:   Fundamentos (AI + Compilers + Stats)
Semana 4-6:   Integración (Projects + Research)
Semana 7-10:  Aplicación y Tesis
```

## Cómo Usar Este Material

- Cada módulo tiene lecturas secuenciales numeradas
- Los bloques de código Python son ejecutables donde está indicado
- Las referencias cruzadas conectan conceptos entre módulos
- Los ejercicios al final de cada lectura refuerzan el aprendizaje
