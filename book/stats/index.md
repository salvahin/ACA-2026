# Estadística y Métodos Cuantitativos

Este módulo te proporciona las herramientas estadísticas necesarias para diseñar experimentos rigurosos, analizar resultados y reportar hallazgos de manera profesional. Aprenderás desde fundamentos de probabilidad hasta técnicas avanzadas de análisis, con énfasis en reproducibilidad científica.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

- [ ] Aplicar conceptos de probabilidad y distribuciones a problemas de ML
- [ ] Diseñar experimentos con poder estadístico adecuado
- [ ] Seleccionar y ejecutar pruebas de hipótesis apropiadas
- [ ] Calcular e interpretar tamaños de efecto e intervalos de confianza
- [ ] Implementar prácticas de reproducibilidad en tus experimentos
- [ ] Reportar resultados estadísticos siguiendo estándares académicos

## Prerequisitos

- Programación en Python
- Familiaridad con NumPy/Pandas
- Conceptos básicos de Machine Learning
